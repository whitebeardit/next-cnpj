# next-CNPJ

Biblioteca .NET para validação robusta de CNPJ conforme NTC 2025.001, com suporte ao formato numérico tradicional e ao novo formato alfanumérico.

## Build e Testes

- Build: `dotnet build`
- Testes: `dotnet test`

## Framework

- Target: .NET Standard 2.0
