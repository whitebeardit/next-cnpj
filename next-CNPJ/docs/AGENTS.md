# next-CNPJ

.NET library for robust CNPJ validation according to NTC 2025.001, with support for traditional numeric format and the new alphanumeric format.

## Build and Tests

- Build: `dotnet build`
- Tests: `dotnet test`

## Framework

- Target: .NET Standard 2.0
